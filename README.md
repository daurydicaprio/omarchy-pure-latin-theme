# 🎨 Pure-Latin Theme for Omarchy

A dark, minimalist theme for the Omarchy Arch Linux remix. Designed for a clean, professional, and low-contrast visual experience, ideal for long coding or gaming sessions without eye strain. The aesthetic is built upon a foundation of deep blue-greys with subtle, cool-toned accents.

Created by **Daury Dicaprio**.

---

### ✨ Features

-   **Cohesive Palette:** A consistent color scheme applied across all major components.
-   **Professional Feel:** A low-contrast, minimalist design that prioritizes focus.
-   **Dynamic Elements:** Includes a subtle gradient for active window borders.
-   **Complete Integration:** Styles the Terminal, Status Bar, Menus, Notifications, and more.

### 🎨 Color Palette

| Role               | Hex       | Description                              |
|--------------------|-----------|------------------------------------------|
| **Background**     | `#262A33` | A dark, cool charcoal grey with a blue hue.  |
| **Foreground**     | `#D3DEE9` | A soft, off-white for excellent readability. |
| **Subtle Accent**  | `#333842` | A very dark blue-grey for borders and accents. |
| **Neutral Grey**   | `#404040` | A pure, mid-tone grey for neutral elements.    |
| **Primary Accent** | `#5E81AC` | A muted, professional dark blue.           |

---

### 🚀 Installation

Use the built-in Omarchy theme installer:

```sh
omarchy-theme-install https://github.com/daurydicaprio/omarchy-pure-latin-theme
```

After installation, select **`pure-latin`** from the Omarchy theme menu (`Super + Space` -> `Style` -> `Pick new theme`).

### 🛠️ Components Styled

This theme provides a consistent look and feel across the entire Omarchy desktop environment by styling the following components:

-   **Terminal (`Alacritty`):** A full 16-color palette.
-   **Status Bar (`Waybar`):** Minimalist color definitions.
-   **Window Manager (`Hyprland`):** Gradient on the active window border.
-   **Application Menus (`Walker`):** Styled to ensure perfect contrast.
-   **System Monitor (`btop`):** A custom monochromatic theme.
-   **Notifications (`Mako`):** Clean notification pop-ups.
-   **Code Editor (`Neovim`):** Configured to use a matching `colorscheme`.
