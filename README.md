# Pure-Latin Theme for Omarchy

A dark, minimalist theme for the Omarchy Arch Linux remix, designed for a clean, professional, and low-contrast visual experience. Ideal for long coding or gaming sessions without eye strain, the aesthetic is built upon a foundation of deep blue-greys with subtle, cool-toned accents.

---

### ✨ Features

-   **Cohesive Palette:** A consistent and subtle color scheme applied across all major system components.
-   **Professional Feel:** A low-contrast, minimalist design that prioritizes focus and reduces visual clutter.
-   **Dynamic Borders:** Includes a subtle, directional gradient on the active window border for clear focus without distraction.
-   **Complete Integration:** Styles the Terminal, Status Bar, Menus, Notifications, System Monitor, and more for a unified desktop experience.

### 🎨 Color Palette

| Role               | Hex       | Description                              |
|--------------------|-----------|------------------------------------------|
| **Background**     | `#262A33` | A dark, cool charcoal grey with a blue hue.  |
| **Foreground**     | `#D3DEE9` | A soft, off-white for excellent readability. |
| **Subtle Accent**  | `#333842` | A very dark blue-grey for borders and accents. |
| **Primary Accent** | `#5E81AC` | A muted, professional dark blue.           |

---

### 🚀 Installation

Use the built-in Omarchy theme installer. Simply run the following command in your terminal:

```sh
omarchy-theme-install https://github.com/daurydicaprio/omarchy-pure-latin-theme
```

After installation, select **`pure-latin`** from the Omarchy theme menu (`Super + Space` -> `Style` -> `Pick new theme`).

---

### 🛠️ Components Styled

This theme provides a consistent look and feel by styling the following applications and components:

-   **Terminal:** `alacritty.toml`
-   **Status Bar:** `waybar.css`
-   **Window Manager:** `hyprland.conf`
-   **Application Menus:** `walker.css`
-   **Notifications:** `mako.ini`
-   **System Monitor:** `btop.theme`
-   **Code Editor:** `neovim.lua`
-   **On-Screen Display:** `swayosd.css`
-   **Lock Screen:** `hyprlock.conf`
